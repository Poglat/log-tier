## Foreword

Log Tier is quite the small project, and I am the sole maintainer. As such not all requests will be fulfilled, and the repo may be archived in future. If your work can't be added, I encourage you to fork the repo and work on it yourself.

I will do my best to review all contributions in a timely manner, but I can't guarantee yours will be.

Log Tier is licensed under the [GPL-3.0 only](LICENSES/gpl-3.0.txt), by contributing to the repo, you agree that your work can, and will be licensed under the same license.

## Issues

If when using Log Tier you find an error, or the potential for a new feature, feel free to raise an issue to the repo.

## Pull Requests

If you find an error, or the potential for a new feature, and wish to implement your own changes to the main repo, please create a pull request with your desired changes. 

To create a pull request

1. Fork the repo
2. Clone the forked repo
3. Make a branch, and push changes to your fork
4. Open a PR on the main repo, using your forked branch

Please ensure you've adequately tested any branch you work on before making the pull request.
